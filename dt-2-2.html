线性表（一）

发表于 2017-11-13 | 分类于 Demo
数据结构中，一对一的关系叫做表。表的顺序表示指的是使用连续的存储单元存储数据。这种结构的优点在于能够快速定位，查找，搜索速度快，缺点在于插入删除非常慢。

0x00

大二的数据结构，一开始是用 C 写的，今天讲了树，感觉继续用 C 写的话，越往后（抽象程度越高）会越难受，干脆趁早换成 C++ ，那么从头把线性表再写一遍吧，以后估计还要用到。(简单测试了一下，应该是没错误的…)(很想学习一下gtest单元测试框架，但是懒啊)(想用 python 或 js ，觉得老师不会同意…)

一些简单说明：
assert(exp) 表示断言，当 exp 表达式为假时，则抛出 Assertion failed (断言失败) ；
bool (*compare)(int, int) 表示定义一个函数指针 compare ，它有两个 int 参数，返回值为 bool 型；
